Toys api documentation

Routes

All HTTP methods are supported. You can use http or https for your requests.

Get toys

  • Get products:/toys (Get)
  • Search products by name or info:/toys/?s=key word (Get)
  • Get 10 products by page number:/toys/?page=number (Get)
  • Get all products by category:/toys/cat/:categoryName (Get)

    • options of categoryName: Cars, Construction games, Tabletop games.

  • Price range:/toys/prices/?min=10&max=40 (Get)

Add toy

Get toy - (only for register user - must send user's auth-token through headers):/toys (Post)
  • name (required)
  • info (required)
  • price (required)
  • category (required)
  • img_url

Edit toy

Edit toy - (only for register user - must send user's auth-token through headers):/toys/:_id (toy) (PUT)
  • name (required)
  • info (required)
  • price (required)
  • category (required)
  • img_url

Delete toy

Delete toy - (only for register user - must send user's auth-token through headers): /toys/:_id (toy) (DELETE)

Sign Up

Sign Up:/users (Post)
  • name (required)
  • email (required)
  • password (required)
  • role

Delete user

Delete user - (only for register user - must send user's auth-token through headers): /users/:_id (user) (DELETE)

Login user

Login:/users/login (Post)
  • email (required)
  • password (required)
